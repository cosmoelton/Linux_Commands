###### Comando cd

$ cd – change directory

O comando “cd” é utilizado para navegar entre diretórios

###### Comando ls
$ ls – list directory
O comando “ls” é utilizado para mostrar arquivos e diretórios existentes no seu disco, bem como suas propriedades.
